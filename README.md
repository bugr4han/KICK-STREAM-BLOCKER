  <h3 align="center">KICK STREAM BLOCKER</h3>
  
  <p align="center">
    Kick Stream Blocker is a lightweight and privacy-friendly Chrome extension that helps you filter and hide unwanted streams
    on the <strong>kick.com/browse</strong> page by category, title keywords, and streamer names.
    With real-time filtering and dynamic content support, blocked streams are removed instantly without any page refresh.
  </p>

  <p align="center">
    All settings are stored securely using Chrome Sync and work seamlessly across your devices.
    Built for users who want a cleaner, faster, and distraction-free browsing experience on Kick.
  </p>

</div>

---

<div align="center">
  <h3 align="center">HOW TO DOWNLOAD & USE</h3>
</div>

---

<div align="center">
  <p align="center">
    You can install and start using Kick Stream Blocker by following the methods below.
    Both options are safe and officially supported.
  </p>

  <p align="center">
    <strong>1. Download from GitHub Releases</strong><br/>
    Go to the <strong>Releases</strong> section of this repository, download the <strong>KICK-STREAM-BLOCKER (ZIP)</strong> file,
    extract it to any folder on your computer, then follow the manual installation steps below.
  </p>

  <p align="center">
    <strong>2. Manual Installation (Developer Mode)</strong><br/>
    Open <code>chrome://extensions</code> in your browser, enable <strong>Developer Mode</strong>,
    click <strong>Load unpacked</strong>, and select the extracted project folder.
    The extension will be installed instantly and ready to use.
  </p>

  <p align="center">
    After installation, click the extension icon, enable the switch, and start adding your own filter rules
    by category, title keyword, or streamer name. All changes are applied instantly.
  </p>
</div>
